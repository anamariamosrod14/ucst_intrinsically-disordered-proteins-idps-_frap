# Data folder

Place raw FRAP Excel exports in `data/raw/`.

Expected Excel columns:

- `Adjusted time [s]`
- `Intensity Region 1`
- `Intensity Region 2`
- `Intensity Region 3`

Region definitions:

- Region 1 = pore ROI
- Region 2 = matrix / continuous phase ROI
- Region 3 = unbleached reference ROI

If raw Excel files are too large or cannot be shared publicly, include processed outputs in `data/processed/` and note that raw data are available upon request.
