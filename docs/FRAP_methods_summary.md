# FRAP methods summary

FRAP experiments were analyzed using double normalization against an unbleached reference ROI. Pore and matrix ROIs were processed separately.

The analysis reports:

- `t_half_s`: recovery half-time extracted from normalized recovery curves.
- `D_app_um2_s`: apparent diffusion coefficient estimated using the Soumpasis relation.
- `mobile_fraction`: mobile fraction calculated as `(Finf - F0)/(1 - F0)`.
- `K`: relative diffusivity calculated as `D_app/D_PBS`.

Because the hydrogels contain heterogeneous pore and matrix microenvironments, the diffusion coefficients are reported as apparent diffusion coefficients.
